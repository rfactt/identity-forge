# Identity Forge

Identity Forge é uma ferramenta feita em Python para gerar usernames de forma criativa com base no nome do usuário,
 tema escolhido e estilo de identidade.

## Funcionalidades
- Geração de usernames únicos
- Estilos personalizados:
  - hacker
  - clean
  - profissional
  - criativo
- Escolha da quantidade de sugestões
- Validação de entradas
- Loop interativo no terminal
- Limpeza de tela a cada nova execução

## Tecnologias usadas
- Python
- random
- os

## Como executar

```bash
python identityforge.py
